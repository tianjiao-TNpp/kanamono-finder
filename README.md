# 五金件さがし / 金物・締結部品ファインダー

部品の名前が分からなくても、**「何をしたいか」** から金物（五金件・締結部品）を見つけられる日本語ツールです。
単体HTML・オフライン動作。ブラウザで開くだけで使えます。

## 公開リンク
GitHub Pages: （リポジトリ作成後に追記）

## 特長
- 🧭 **用途から探す（診断式）** — 材料・状況をタップで辿り、候補と「なぜそれか」を提示
  - 重さ・サイズで提案が変わる **2段階の質問** に対応
- 🔍 **キーワード検索** — 名前・通称・特徴のフリーワードで検索（カタカナ/ひらがな・部分一致対応）
- 各部品に **イラスト（SVG）** と **サイズ表（M6×20 等の目安）** を表示
- 収録 **94種**／診断カテゴリ **16**（締結・釘・建築金物・建具/錠前・配管/電設・チェーン/ロープ・ゴム/ばね など）

## 使い方
`index.html` をブラウザで開くだけ。サーバー不要。

## カスタマイズ
- 部品を増やす: `index.html` 内の `DB = [...]` に1件追加（`name / yomi / cat / pic / aliases / desc / uses / spec / tags`）
- 用途診断を増やす: `INTENTS = [...]` に質問・選択肢・推奨部品(`recs:[{id, why}]`)を追加
- 実写真に差し替え: `images/` に写真を置き、該当部品に `img:"images/xxx.jpg"` を追加（未設定はSVG表示）

## 注意
サイズ・荷重は一般的な目安です。最終的な選定は耐荷重・材質・規格をご確認ください。
